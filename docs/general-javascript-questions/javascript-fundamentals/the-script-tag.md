### What is the “script” tag used for in JavaScript?

<details>
  <summary>
    <strong>View Answer:</strong>
  </summary>
  <div>
    <div>
      <strong>Interview Response:</strong> We can use the script tag to insert JS into HTML or import scripts from external sources.
    </div>
    <br />
    <div>
      <strong>Technical Response:</strong> JavaScript may be included practically anywhere in an HTML page using the script tag—the script tag we use to import scripts from external file sources.
    </div>
    <div>
    <br />
    <strong class="codeExample">Code Example:</strong><br /><br />
    <div></div>

```javascript
<script src="/path/to/script.js"></script>
// External URL:
<script src="https://cdnjs.cloudflare.com/ajax/libs/lodash.js/4.17.11/lodash.js"></script>
```
